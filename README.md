---
## Estructura del repositorio

El material esta organizado por lenguajes y prácticas, facilitando la navegación:

* **`/haskell`**: Contiene las prácticas y ejercicios del lenguaje Haskell.
    * `/Practica1`, `/Practica2`, etc.
* **`/prolog`**: Incluye el material y las guías de Prolog.
* **`/wollok`**: Dedicado a las prácticas del lenguaje Wollok con su enunciado
* **`/Finales`**: Posibles soluciones a los finales

## Cómo Colaborar?

Todo aporte es bienvenido y esencial para mantener este material actualizado Si tenes una mejor solución, otra alternativa de resolución, o has detectado un error, te invitamos a contribuir

### Pasos para Colaborar:

1.  **Haz un Fork** del repositorio.
2.  **Clona** tu copia local.
3.  **Crea una nueva rama** para tu aporte (`git checkout -b feature/mi-nuevo-aporte`).
4.  **Realiza tus cambios** y **haz commit** (`git commit -m "feat: Agrego solucion alternativa a Practica X"`).
5.  **Sube los cambios** a tu fork (`git push origin feature/mi-nuevo-aporte`).
6.  **Abre un Pull Request (PR)** hacia este repositorio.

Si queres ser agregado como colaborador directo, o simplemente practicar y subir tus soluciones como modo de practica, simplemente abre un Issue presentándote y mencionando cómo te gustaría contribuir y te agregamos sin drama :P
